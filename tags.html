Tags

  • The <h1> element defines a heading. - <h1>My First Heading</h1>
  • The <nav> element defines a container for navigation links. - <nav>My Navigation bar</nav>
  • The <header> element defines a header for a document or a section. - <header>My First Header</header>
  • The <p> element defines a paragraph. - <p>My First Paragraph</p>
  • The <footer> element defines a footer for a document or a section. - <footer>My First Footer</footer>
